【Go学习笔记】类型

2019年8月15日 15:53

1.1 变量

  • var 定义变量
  • 静态类型，不能在运⾏行期改变变量类型
  • 自动初始化为零
  • 初始化类型推断
  • 多变量定义
  • 多变量赋值
  • 占位变量_
  • 未使用局部变量视为错误
  • 区分重新赋值和重新定义
var x int // 自动初始化 0
var y = 1 // 自动类型推断 int
var (
    a int = 2
    b float32 = 3.14
    c string = "test"
    d bool = true
) // 多变量定义

func main(){
    a, b, d = 3, 3.1415, false // 多变量赋值
    a := "test" // 重新定义
    fmt.Println(a)
}

1.2 常量

  • const定义常量
  • 常量值必须是编译期可确定的数字、字符串、布尔值
  • 不支持 1UL、2LL 这样的类型后缀
  • 常量组中，如不提供类型和初始化值，那么视作与上⼀一常量相同
  • 可以是len、cap、unsafe.Sizeof 等编译期可确定结果的函数返回值
  • 关键字 iota 定义常量组中自增枚举值
  • 自定义枚举类型
const (
    X, Y = 2, true
    Xcopy, Ycopy
    A, B = iota, iota << 10 // 0, 0 << 10
    C, D // 1, 1 << 10
)

1.5 类型转换

  • 不支持隐式类型转换
  • 使用括号避免优先级错误
  • 不能将其他类型当 bool 值使⽤

1.6 字符串

  • 双引号”字符串”，单引号rune’我’，反引号`不转义跨行`
  • 不可变值类型，指向UTF-8字节数组
  • 默认空字符串”“而非nil
  • &s[i]非法
  • s[i] = c非法
  • `abc\r\n\x00` 不转义跨行字符串
  • 连接字符串时 “+” 必须在行末尾
  • 支持s[1:5]
  • 转换成[]byte或[]rune，修改string
  • 按byte遍历 / 按rune遍历
func main() {
    s := "abc汉字"
    for i := 0; i < len(s); i++ { // byte
        fmt.Printf("%c,", s[i])
    }

    for _, r := range s { // rune
        fmt.Printf("%c,", r)
    }
}

1.7 指针

func main() {
    d := struct {
        s string
        x int
    }{"abc", 100}
    p := uintptr(unsafe.Pointer(&d)) // *struct -> Pointer -> uintptr
    p += unsafe.Offsetof(d.x) // uintptr + offset
    p2 := unsafe.Pointer(p) // uintptr -> Pointer
    px := (*int)(p2) // Pointer -> *int
    *px = 200 // d.x = 200
    fmt.Printf("%#v\n", d)
}
